## Studio Pro

Painel de planejamento de conteudo pensado para rodar como site estatico.

## Stack atual

- HTML
- CSS
- JavaScript puro
- localStorage para persistencia local

## Recursos

- Cadastro e edicao de cards
- Favoritos, filtros e metricas
- Metas de performance
- Upload de thumbnail em Base64
- Backup e restauracao por JSON
- Tema salvo no navegador

## Como usar localmente

Abra [src/frontend/index.html](src/frontend/index.html) no Live Preview da sua IDE.


## Observacao

Os dados ficam salvos no navegador e no dispositivo em que o site for usado.
